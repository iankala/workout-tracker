# 💪 Workout Tracker

A full-stack workout tracking application with a JavaScript frontend and a Node.js backend.

## About

Workout Tracker lets users log and monitor their workouts. It features a separated frontend and backend architecture, making it a great example of a full-stack JavaScript project.

## Features

- Log and track workouts
- Full-stack architecture (frontend + backend)
- RESTful backend API
- Persistent data storage

## Tech Stack

- **Frontend** — JavaScript, HTML, CSS
- **Backend** — Node.js
- **JavaScript** — Shared language across the stack

## Getting Started

### Backend

1. Clone the repository:
   ```bash
   git clone https://github.com/iankala/workout-tracker.git
   cd workout-tracker/backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the backend server:
   ```bash
   npm start
   ```

### Frontend

1. Navigate to the frontend folder:
   ```bash
   cd ../frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the frontend:
   ```bash
   npm run dev
   ```
